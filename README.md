# Usage
<!-- usage -->
```sh-session
$ npm install -g node-red-dev
$ node-red-dev COMMAND
running command...
$ node-red-dev (-v|--version|version)
node-red-dev/0.0.1 darwin-x64 node-v14.15.3
$ node-red-dev --help [COMMAND]
USAGE
  $ node-red-dev COMMAND
...
```
